# adguardcli-script
My bash script for adguard-cli on linux because im to lazy to remember certain commands.
it installs the latest version if Adguard-CLI cacn't be found on the system.

## Images

![Main Menu](https://github.com/CodingAlphaWolf/adguardcli-script/blob/main/pictures/adgcli.png)
![Filter Section of Menu](https://github.com/CodingAlphaWolf/adguardcli-script/blob/main/pictures/filtermenu.png)

## Features
* Menu driven interactions with Adguard-CLI
* Option to inport and export settings.
* Start or stop the service.
* A filter section where you can update, remove, change and see your current filters.
* Auto update check for Adguard itself (Not sure if its working)
